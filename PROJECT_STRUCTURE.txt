================================================================================
                         KEY FILES EXPLAINED
================================================================================

BACKEND ENTRY POINT:
  backend/app.py
  - Main FastAPI application
  - Registers all route routers
  - Sets up CORS middleware
  - Configures MongoDB connection
  - Serves frontend static files

CORE RECOMMENDATION ENGINE:
  backend/services/recommendation/onet_loader.py
  - Loads O*NET data (879 IT careers)
  - Filters to IT-only occupations
  - Pre-computes embeddings

  backend/services/recommendation/recommendation_engine.py
  - Matches user skills to careers
  - Uses sentence-transformers
  - Returns top N matches with scores

SKILL GAP ANALYSIS:
  backend/services/skill_gap/gap_analysis.py
  - Compares user skills to target role
  - Identifies missing skills
  - Calculates completion percentage

  backend/services/skill_gap/learning_recommender.py
  - Uses Groq LLM to generate recommendations
  - Provides free learning resources
  - Estimates learning time

FRONTEND ENTRY POINT:
  frontend/index.html
  - Standalone HTML UI
  - Skill selector
  - Career recommendations display
  - Skill gap visualization
  - Learning resources section
  - Resume upload interface

API ROUTES:
  backend/routes/recommendation_routes.py
  - POST /api/recommendations/

  backend/routes/dashboard_routes.py
  - POST /api/dashboard/

  backend/routes/profile_routes.py
  - POST /api/profile/skill-gap
  - POST /api/profile/learning-path

  backend/routes/resume_routes.py
  - POST /api/resume/upload

  backend/routes/trend_routes.py
  - POST /api/trends/

  backend/routes/auth_routes.py
